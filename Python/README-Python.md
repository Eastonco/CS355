## Getting Started

### Prerequisites

Install Python3

## Running the program 
Here's a list of handy commands: to run tests

```
python -m unittest HW3SampleTests.py
```
or if you want to screw around with your own tests, open python with the interactive flag -i

```
python -i HW3.py
```
to quit 
```
CRTL + D
```
